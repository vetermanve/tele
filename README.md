# wallet.tg P2P Monitor

Терминальный мониторинг P2P-цен на [wallet.tg](https://wallet.tg) (Telegram Wallet) в реальном времени.
Показывает лучшие объявления на продажу **USDT/RUB** и **TON/RUB** с отклонением от рыночного курса.

Живой TUI на `prompt_toolkit` + `rich`: цены обновляются каждые 20 секунд, новый лидер по цене подсвечивается миганием.

## Возможности

- Две пары сразу: `USDT/RUB` и `TON/RUB`.
- Рыночный курс с фолбэком по источникам: **CoinGecko → Binance → CryptoCompare**, с кэшем на 10 минут (`.rates_cache.json`).
- Колонка отклонения P2P-цены от рынка с цветовой градацией (`<1%`, `1-3%`, `3-5%`, `>5%`).
- Интерактивные фильтры прямо в TUI (сохраняются в `.filters.json`):
  - диапазон суммы сделки (MIN / MAX ₽),
  - минимальный рейтинг мейкера,
  - чёрный список способов оплаты (объявление скрывается, только если **все** его способы исключены).
- Отсев мелких мейкеров (`< 50` сделок) и подсветка онлайн/офлайн.

## Установка

```bash
git clone git@github.com:vetermanve/tele.git
cd tele

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Настройка

Нужен ключ от wallet.tg P2P integration-api. Положи его в `.env`:

```bash
cp .env.example .env
# отредактируй .env, подставь свой WALLET_TG_API_KEY
```

`.env` в `.gitignore` и в репозиторий не попадает. Скрипт подхватывает его через `python-dotenv`.
Ключ также можно задать переменной окружения:

```bash
export WALLET_TG_API_KEY=...
```

## Запуск

```bash
python monitor.py
```

### Управление

| Клавиша            | Действие                                         |
|--------------------|--------------------------------------------------|
| `↑` / `↓`          | переключить активное поле фильтра                 |
| число + `Enter`    | задать значение поля (MIN / MAX / рейтинг)        |
| `0` + `Enter`      | сбросить поле                                    |
| `Ctrl+C` / `Ctrl+Q`| выход                                            |

### Поле «Искл. оплату»

Выбери поле `Искл. оплату` через `↑`/`↓` и введи название или код способа оплаты + `Enter` — метод добавится в чёрный список (повторный ввод убирает его). `0` + `Enter` очищает список.

- Принимаются и русские названия (`Сбер`, `Тинькофф`, `Душанбе Сити`), и сырые коды из API (`international_transfer`).
- Объявление с несколькими способами (напр. `СБП + Сбер`) скрывается только когда **все** его способы в чёрном списке — пока остаётся хоть один годный, объявление видно.

## Зависимости

- `requests` — HTTP-запросы к API.
- `rich` — таблицы и цветной вывод.
- `prompt_toolkit` — интерактивный полноэкранный TUI.
- `python-dotenv` — загрузка ключа из `.env`.

## Структура

```
monitor.py        — весь код: поллинг, рендеринг, TUI
requirements.txt  — зависимости
.env.example      — шаблон конфигурации
.filters.json     — сохранённые фильтры (создаётся на лету, в .gitignore)
.rates_cache.json — кэш курсов (создаётся на лету, в .gitignore)
```

## Замечания

- Эндпоинты публичных бирж иногда лимитируют запросы — для этого и сделан кэш + цепочка фолбэков.
- Данные носят информационный характер, это не торговый инструмент.
